## 插件开发

#### 安装 ToolDelta 库

你需要使用 Python 3.10+ 配置 ToolDelta 的插件开发环境。
使用 pip 安装 ToolDelta：
```sh
pip install tooldelta
```

进入 ToolDelta 工作目录， 即可进入插件开发环境！

目前 ToolDelta 仅支持类式插件开发。

- 进入 [类式插件开发教程](./class-plugin/创建插件)

- 进入 [类式插件开发实战](./class-plugin/demo/示例插件.md)

在浏览文档的时候，我们强烈建议通过 **侧边栏** 作为导航栏浏览，点击左上角的 **菜单** 按钮即可直接进入接口导航栏查看接口文档。

什么？读文档太烦人了？可以去 [**插件市场**](/use/plugin_market) 下载几个插件作为学习模版！
